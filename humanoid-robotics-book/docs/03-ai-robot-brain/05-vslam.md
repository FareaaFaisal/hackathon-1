## VSLAM (Visual SLAM)

This chapter focuses on map creation and localization using Visual SLAM.

### Requirements:
- Map creation.
- Localization.

### Success Criteria:
- Robot performs VSLAM in Isaac Sim.

---
**Learning Outcomes**:
- Understand the principles and algorithms behind Visual SLAM.
- Implement VSLAM techniques for real-time map creation and robot localization.
- Utilize Isaac Sim to simulate VSLAM performance in various environments.
