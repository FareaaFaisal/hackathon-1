## Gazebo Setup & Fundamentals

This chapter covers the installation and fundamental usage of Gazebo.

### Requirements:
- Installation.
- World files.

### Success Criteria:
- Student loads a simple world.

---
**Learning Outcomes**:
- Install and configure Gazebo.
- Understand Gazebo's interface and core functionalities.
- Load and interact with basic simulation environments (world files).
